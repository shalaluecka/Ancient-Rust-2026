**Ancient Rust Toolbox** is a comprehensive, open-source companion utility (.exe) engineered to enhance your training and gameplay experience in Rust. Created for both competitive players and content creators, this tool provides a safe, fully external suite of features ranging from spray pattern visualization to localized game configuration management—without modifying game memory.

This utility operates completely outside of the game process as an isolated overlay and configuration writer. It does not perform memory injection, DLL hooking, or engine tampering, making it entirely compliant with global anti-cheat standards (EAC).

> **Important Disclaimer:** Ancient Rust Toolbox is an external assistant tool designed exclusively for training server practice, strategic planning, and configuration optimization. It does not automate gameplay or violate the game's terms of service.

---

## Key Features

### 1. Recoil Pattern Analysis & Training Mode
* **Visual Spray Guides:** Displays accurate, static geometric patterns of weapon recoil directly on your secondary monitor or overlay to assist with mechanical muscle memory training.
* **Sensitivity Calibrator:** Helps calculate the perfect in-game mouse sensitivity based on your DPI for maximum consistency during spray control practice.

### 2. Advanced Performance & Config Optimizer
* **Client FPS Booster:** Easily modify hidden game parameters within your local `client.cfg` to maximize frames per second (FPS) and decrease input lag on low-end systems.
* **Network & Ping Stabilization:** Tweaks connection parameters to optimize network packet sending intervals, resulting in a smoother experience on high-population servers.

### 3. Tactical Overlay & Tools
* **Custom Crosshair Engine:** A safe, hardware-accelerated external overlay that draws a customizable crosshair on your screen (perfect for monitoring hip-fire accuracy).
* **Raid & Eco Calculator:** Built-in resource module to quickly calculate the exact sulfur, charcoal, and explosive costs required for any raiding scenario.

## Installation & Deployment Guide

1. Navigate to the **Releases** tab on the right side of this repository page and download the latest compiled binary (`AncientLoader Setup`).
2. Open your standard game configuration or launch Rust to your preferred training server.
3. Run the executable file as an Administrator to ensure the overlay can properly render above background window layers.
4. Select your desired module (e.g., Crosshair Designer, Config Tuning, or Spray Pattern Viewer) from the clean graphical interface.
5. Apply your settings and enjoy a more optimized, structured training environment.
